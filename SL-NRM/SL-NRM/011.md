# 1. CORE API FEATURES (CÓ SẴN TRONG LARAVEL)
Đây là các tính năng **có sẵn ngay lập tức** sau khi tạo project, bạn chỉ cần dùng lệnh `php artisan make:...` để tạo ra dùng. KHÔNG CẦN cài đặt thêm gì cả.

## 1. Form Requests (Validate Dữ liệu) -> *Cái bạn đang hỏi*
Thay vì viết `if ($request->missing('email'))` trong Controller request nào cũng phải check, Laravel có sẵn class này.
- **Tác dụng**: Tự động kiểm tra dữ liệu đầu vào. Nếu sai -> Tự động trả về JSON lỗi 422 cho API.
- **Lệnh tạo**: `php artisan make:request StoreProductRequest`
- **Logic**: Viết trong file `app/Http/Requests/StoreProductRequest.php`.

## 2. API Resources (Format JSON)
FE React cần JSON đẹp, chuẩn, không lộ password.
- **Tác dụng**: Biến Model (Database) thành JSON (API Response).
- **Lệnh tạo**: `php artisan make:resource ProductResource`
- **Logic**: Viết trong file `app/Http/Resources/ProductResource.php`.

## 3. Eloquent API Resources (Query Database)
- **Tác dụng**: Các hàm có sẵn để query DB nhanh.
- **Tính năng**: `Model::with()`, `Model::paginate()`, `Model::create()`.
- **Lệnh tạo Model**: `php artisan make:model Product -m`

## 4. Gates & Policies (Phân quyền cơ bản)
- **Tác dụng**: Kiểm tra xem User này có được Sửa/Xóa cái Đơn hàng kia không.
- **Lệnh tạo**: `php artisan make:policy OrderPolicy`
- **Logic**: Viết trong `app/Policies/OrderPolicy.php`.

## 5. API Routes
- **Tác dụng**: Định nghĩa đường dẫn API (endpoint).
- **Vị trí**: File `routes/api.php` có sẵn.

## 6. Exception Handling (Xử lý lỗi)
- **Tác dụng**: Khi code lỗi, Laravel tự trả về JSON `{"message": "Server Error"}` thay vì trang HTML lỗi.
- **Cấu hình**: File `bootstrap/app.php`.

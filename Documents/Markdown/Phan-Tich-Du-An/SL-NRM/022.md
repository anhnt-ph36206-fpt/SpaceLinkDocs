# 2. EXTERNAL LIBRARIES (CÀI THÊM TỪ BÊN NGOÀI)
Đây là các thư viện **không có sẵn**, muốn dùng phải chạy lệnh `composer require ...`.

## 1. Laravel Sanctum (Đã cài via install:api)
- **Loại**: Authentication (Xác thực).
- **Tác dụng**: Tạo Token đăng nhập cho ReactJS (`Bearer ...`).
- **Lệnh cài**: `composer require laravel/sanctum`
- **Phục vụ**: Chức năng Đăng nhập/Đăng ký.

## 2. Spatie Laravel Permission (Cài thêm)
- **Loại**: Authorization (Phân quyền nâng cao).
- **Tác dụng**: Quản lý Role & Permission chuyên nghiệp (thay vì code tay bảng role_permissions).
- **Lệnh cài**: `composer require spatie/laravel-permission`
- **Lưu ý quan trọng**: Library này **KHÔNG KHỚP** với file SQL `new-claude-sl_db.sql` hiện tại của bạn (Library này tự tạo bảng riêng).
    - *Option A*: Sửa file SQL để dùng bảng của Spatie.
    - *Option B*: Không cài Spatie, dùng Gate/Policy của Core (như file 01).
- **Khuyên dùng**: Nếu bạn "lười" code logic phân quyền -> Cài Spatie và bỏ các bảng Role tự tạo đi.

## 3. Maatwebsite Excel (Cài thêm)
- **Loại**: Utility (Tiện ích).
- **Tác dụng**: Xuất/Nhập file Excel.
- **Lệnh cài**: `composer require maatwebsite/excel`
- **Phục vụ**: Chức năng #20 Thống kê & Xuất báo cáo.

## 4. L5-Swagger hoặc Scramble (Cài thêm)
- **Loại**: Documentation (Tài liệu).
- **Tác dụng**: Tự sinh trang web document API cho team FE đọc.
- **Lệnh cài**: `composer require khulnasoft/scramble` (Khuyên dùng vì nhẹ hơn Swagger).

## 5. GuzzleHTTP (Thường có sẵn nhưng là Lib ngoài)
- **Loại**: HTTP Client.
- **Tác dụng**: Gọi API thanh toán Momo/VNPay.
